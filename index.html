Ruby: Non-Visual Computer Interface

Ruby is a voice-activated, non-visual computer interface designed to provide full system accessibility for blind and visually impaired users without the need for a traditional monitor or screen.

Installation Package

Download the ZIP file below, extract it, and run install.bat as an Administrator.

Download Ruby Installation (dist.zip)

Quick Start Instructions

  • Step 1: Right-click the downloaded ZIP and select "Extract All".
  • Step 2: Open the extracted folder and find install.bat.
  • Step 3: Right-click install.bat and select Run as Administrator.
  • Step 4: Once finished, press Ctrl + Shift + Left-Click to wake Ruby.
